# Apache2-PHP-for-Laravel
Apache2 + PHP Bash installer for Laravel Setup

```
apt update && apt upgrade -y

apt install -y \
    apache2 \
    php \
    php-cli \
    php-common \
    php-mysql \
    php-curl \
    php-gd \
    php-mbstring \
    php-xml \
    php-bcmath \
    php-zip \
    php-intl \
    php-readline \
    php-dev \
    php-imap \
    php-opcache \
    php-soap \
    php-tokenizer \
    php-xsl \
    php-fpm \
    libapache2-mod-php
```
